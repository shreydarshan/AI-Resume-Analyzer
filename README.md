# AI Resume Analyzer

A web-based application that analyzes resumes against job descriptions using keyword matching and NLP techniques.

## Features
- Upload PDF resume
- Compare with job description
- Generate match score
- Show missing keywords
- Provide improvement suggestions

## Tech Stack
- Python
- Flask
- HTML, CSS
- pdfplumber

## How to Run
pip install flask pdfplumber
python app.py
